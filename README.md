# 🏠 부동산 이곳에서 (BooE) - Backend

부동산 정보를 쉽고 빠르게 조회할 수 있는 **부동산 이곳에서 (BooE)** 프로젝트의 백엔드 서버입니다.

## 🛠 Tech Stack
- **Framework**: Spring Boot 3.x
- **Language**: Java 17
- **Database**: MySQL
- **Build Tool**: Gradle
- **API**: 건축물대장(표제부) 공공데이터 API 연동

## ✨ Key Features (개발 예정 및 진행 중)
- **부동산 매물 정보 관리**: 토지 및 건물 상세 정보 제공
- **건축물대장 연동**: 표제부 API를 통한 면적, 구조, 용도 자동 데이터 수집
- **데이터 구조화**:
  - **토지**: 지목, 대지권 비율, 면적 관리
  - **건물**: 구조, 용도, 전용 면적 관리
- **Flutter 연동**: 모바일 앱과의 RESTful API 통신
